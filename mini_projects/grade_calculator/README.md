# Grade Calculator

A python mini project that calculates percentage and overall grade based on user-entered marks.

## Features
- Dynamic number of subjects
- Input validation
- Percentage calculation
- Grade calculation
- Exception handling
- Retry and exit system

## Concepts Used
- If-Else
- Loops
- Lists
- Exception Handling
- User Input
- Built-in Functions

## How to Run
Run the Python file using:

python grade_calculator.py

## Requirements
- Python 3.x

## Grade System

| Percentage | Grade |
|------------|-------|
|     90+    |   A+  |
|     80+    |   A   |
|     70+    |   B   |
|     60+    |   C   |
|     50+    |   D   |
|     40+    |   P   |
|  Below 40  |  Fail |
