# Dice Rolling Simulator

A python mini project that simulates rolling dice and displays detailed statistics of the results.

## Features
- Choose number of dice
- Choose number of turns
- Per roll results with sum
- Final statistics after all rolls
- Custom mode calculation
- Play again option

## Concepts Used
- Loops
- Functions
- Lists
- Random Module
- Statistics Module
- Exception Handling
- User Input

## How to Run
```
python dice_rolling_simulator.py
```

## Requirements
- Python 3.x
