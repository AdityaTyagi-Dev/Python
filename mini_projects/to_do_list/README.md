# To-Do List

A Python mini project that allows users to manage their tasks through a simple menu-driven interface. Tasks are saved to a file and persist after the program closes.

## Features
- Add tasks with timestamp
- View all saved tasks
- Delete tasks with confirmation
- Tasks saved permanently in txt file
- Input validation
- Clean menu system

## Concepts Used
- Lists
- Loops
- Functions
- If-Else
- Exception Handling
- File Handling (TXT)
- Datetime Module
- User Input

## How to Run
Run the Python file using:
```
python to_do_list.py
```

## Requirements
- Python 3.x

## Menu Options
| Option | Action |
|--------|--------|
| 1 | Add Task |
| 2 | View Tasks |
| 3 | Delete Task |
| 4 | Exit |
