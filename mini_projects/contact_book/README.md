# Contact Book

A Python mini project to manage contacts with full CRUD operations.

## Features
- Add contacts with name, phone and email
- View all contacts
- Search contact by name, phone or email
- Edit contact details
- Delete contact with confirmation
- Duplicate contact check with override
- Input validation (international phone numbers, email format)

## Concepts Used
- Dictionaries
- Lists
- Functions
- If-Else
- Loops
- Exception Handling
- Built-in Functions

## How to Run
Run the Python file using:
```
python contact_book.py
```

## Requirements
- Python 3.x

## Menu Options

| Option | Action |
|--------|--------|
|    1   | Add Contact |
|    2   | View All Contacts |
|    3   | Search Contact |
|    4   | Edit Contact |
|    5   | Delete Contact |
|    6   | Exit |
