# Expense Tracker

A Python mini project that allows users to track their daily expenses by category. Data is saved permanently using CSV file storage.

## Features
- Add expenses with category and date
- View all expenses
- Filter expenses by category
- View total spending
- Category wise summary
- Auto date on every expense
- Input validation

## Concepts Used
- Loops
- Functions
- If-Else
- Exception Handling
- File Handling (CSV)
- csv module
- datetime module

## How to Run
```
python expense_tracker.py
```

## Requirements
- Python 3.x
