# Caesar Cipher

A Python mini project that encrypts and decrypts messages using the Caesar Cipher technique.

## Features
- Encrypt a message with a custom key
- Decrypt a message with a custom key
- Supports uppercase and lowercase letters
- Non-alphabet characters (spaces, numbers, symbols) remain unchanged
- Input validation

## Concepts Used
- Strings
- Functions
- Loops
- ASCII values (ord, chr)
- Modulo operator
- Exception Handling

## How to Run
Run the Python file using:
```
python caesar_cipher.py
```

## Requirements
- Python 3.x

## Menu Options

| Option | Action |
|--------|--------|
| 1 | Encrypt a message |
| 2 | Decrypt a message |
| 3 | Exit |

## Sample Output
```
Enter the message: Hello World
Enter the key: 3
Encrypted message:
Khoor Zruog
```
