# To-Do List

A Python mini project that allows users to manage their tasks through a simple menu-driven interface.

## Features
- Add tasks
- View tasks with total count
- Delete tasks with confirmation
- Input validation
- Clean menu system

## Concepts Used
- Lists
- Loops
- If-Else
- Exception Handling
- User Input
- Built-in Functions

## How to Run
Run the Python file using:
```
python to_do_list.py
```

## Requirements
- Python 3.x

## Menu Options
| Option | Action |
|--------|--------|
| 1 | Add Task |
| 2 | View Tasks |
| 3 | Delete Task |
| 4 | Exit |
